These debug commands are originally intended for developers, please be careful when using some of them. 
To use them type the command of choice, e.g. `?edits`, in the search bar. 

Some example commands are:
- `?edits`: See the status of your edits.
- `?debug-info`: Show renderer version, zoom scale and FPS counter in the top left corner of the map.
- `?wiki`: Search results for all elements with Wikipedia links

You can find the complete list with more details in [docs/DEBUG_COMMANDS.md](https://codeberg.org/comaps/comaps/src/branch/main/docs/DEBUG_COMMANDS.md)
